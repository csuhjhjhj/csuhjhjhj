## About

My recent work focuses on graph-enhanced RAG for manufacturing knowledge management, agent-native security analysis, backdoor detection for LoRA-finetuned LLMs, and LLM-enhanced recommendation models.

## Featured Projects

### ManuGraphRAG

Graph-RAG prototype for manufacturing process knowledge management.

- Full-stack Vue 3 + Flask system for rail manufacturing and nuclear-material process scenarios.
- Supports knowledge-grounded QA, process recommendation, graph visualization, entity extraction, and user authentication.
- Uses domain knowledge graphs and LLM-enhanced summaries to make complex process documents searchable and actionable.

### LLM-Backdoor-TED

Research prototype for detecting backdoor behavior in LoRA-finetuned LLMs.

- Uses Topological Evolution Dynamics (TED) over multi-layer hidden states.
- Builds class-conditioned benign trajectory references and scores anomalous representation paths.
- Evaluates backdoor behavior in MITRE ATT&CK threat-intelligence classification.

### Agent Risk Lab

Lightweight security workbench for LLM agents, MCP-style tools, Skills, RAG applications, and multi-agent workflows.

- Scans tool manifests, Skill metadata, RAG configs, and agent settings.
- Reports prompt injection, excessive permission, secret exposure, unsafe execution, and RAG privacy risks.
- Provides a browser workbench, dependency-free Python CLI, and local API server.

### CGAF-LLM

Graph attention fusion with large language models for next-item recommendation.

- Integrates graph attention networks with LLM semantic signals.
- Targets zero-shot and cold-start recommendation scenarios.
- First-author work accepted by IEEE Transactions on Big Data.

### AdapFedRec

Adaptive federated recommendation framework.

- Improves federated recommendation through adaptive client aggregation and sampling.
- Designed for more robust recommendation under heterogeneous client behavior.
- First-author work accepted by Knowledge-Based Systems.
